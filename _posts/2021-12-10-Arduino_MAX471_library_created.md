* Name : MAX471
* Description : 

Small library for the MAX471 current and voltage sensor for the Arduino eco-system

1. Current range     0-3 A
2. Voltage range   3-25 V
3. ATmega328 battery internal reference voltage supported. 

* Author: Gavin Lyons

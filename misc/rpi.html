Project list for Linux based Single Board Computers(SBC):

  1. Display_Lib_RPI : C++. Display library for Linux based SBC's
  2. RpiMotorLib : Python. Motor control library for Raspberry Pi
  3. I2CDetectGL C++. BCM2835 based I2C bus scanner for Raspberry Pi
  4. Raspberrypi_tempmon : Python. System monitor for Raspberry Pi

A list of utilities for GNU/Linux operating systems :

  1. Cylon : TUI to maintain an Arch based Linux distribution.
  2. Cylon-Deb : TUI to maintain a Debian based Linux distribution.
  3. Backupmenu : CLI utility to backup a Linux distribution.
  4. Bashmultitool : Bash shell function library.
  5. Ccryptmenu : TUI wrapper for Ccrypt.
  6. Opml-convert-RSS : Python CLI file converter

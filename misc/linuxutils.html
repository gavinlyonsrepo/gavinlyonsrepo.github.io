A list of utilities for GNU/Linux operating systems :

  1. Cylon : Bash TUI to maintain an Arch based Linux distribution.
  2. Cylon-Deb : Bash TUI to maintain a Debian based Linux distribution.
  3. BackUpMenu : Bash CLI utility to backup files.
  4. BashMultiTool : Bash shell function library.
  5. CcryptMenu : Bash TUI wrapper for Ccrypt.
  6. Opml-convert-RSS : Python CLI file converter
  7. TV Viewer : Python GUI to view TV program details

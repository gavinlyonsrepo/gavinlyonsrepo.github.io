Utilities and applications for Linux operating systems :

  1. Cylon : Bash TUI, maintains an Arch based Linux distribution.
  2. Aegis : Bash TUI, maintains a Debian based Linux distribution.
  3. BackUpMenu : Bash CLI backup utility.
  4. BashMultiTool : Bash shell function library.
  5. Colossus : Python GUI tool, converts ttf fonts into data arrays.
  6. Guardian : Python GUI tool, converts image files into data arrays.
  7. TV Viewer : Python GUI application to view TV program data.

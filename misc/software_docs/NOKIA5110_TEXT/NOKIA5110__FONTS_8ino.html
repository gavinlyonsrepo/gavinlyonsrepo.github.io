Detailed Description

Test file for NOKIA5110_TEXT showing all 9 fonts. 1-9, software SPI.

Note
  1. There are Nine fonts , font one is default,
  2. For a non-default font to work , One line in the library header file
  3. called "NOKIA5110_TEXT_FONT_DATA.h" Must be modified by USER
  4. the Define statement at start of file for Font X must be commented IN.
  5. define NOKIA5110_FONT_X where X is name of font in FONT DEFINE SECTION.
  6. Tests will not work fully if non-default fonts are not commented in, nothing will appear.

  7. For all this example file to work fully all fonts must be commented in

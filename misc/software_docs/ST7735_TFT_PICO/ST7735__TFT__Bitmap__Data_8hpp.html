Detailed Description

Library header file for ST7735_TFT_PICO bitmap data Data file to hold bitmap data.

Author
Gavin Lyons
Note
Not part of core library only used for holding test data, should be omitted in project use or port.

bitmaps in file

  1. ArrowImage : bi-colour 2048 bytes (128x128)
  2. SunTextImage : bi-colour (40x16)
  3. motorImage : 16 bit color 565, 32768 bytes (128x128)
  4. fruitbowl : 24 bit color, 49152 bytes (128x128)
  5. SpriteTest16 : 16 bit color 565 2048 bytes (32x32) with background 0x7E5F or ST7735_LBLUE

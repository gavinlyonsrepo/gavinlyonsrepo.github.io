Detailed Description

OLED driven by SSD1306 controller. font data file 8 fonts.

Author
Gavin Lyons.
  1. Font_One default (FUll ASCII with mods)
  2. Font_Two thick (NO LOWERCASE)
  3. Font_Three seven segment
  4. Font_Four wide (NO LOWERCASE)
  5. Font_Five tiny
  6. Font_Six HomeSpun
  7. Font_Seven big numbers (NUMBERS ONLY + : . ,one size )
  8. Font_Eight Medium numbers (NUMBERS ONLY + : . ,one size)

void ST7789_TFT::TFTPowerDown ( void  )

Call when powering down TFT.

Note
Turns off Display Sets GPIO low and turns off SPI End SPI operations. SPI0 pins P1-19 (MOSI), P1-21 (MISO), P1-23 (CLK), P1-24 (CE0) and P1-26 (CE1) are returned to their default INPUT behavior.

◆ LCDI2CErrorGet()

uint8_t HD44780PCF8574LCD::LCDI2CErrorGet ( void  )

get I2C error Flag

bcm2835I2Creasoncode.

  1. BCM2835_I2C_REASON_OK = 0x00,Success
  2. BCM2835_I2C_REASON_ERROR_NACK = 0x01,Received a NACK
  3. BCM2835_I2C_REASON_ERROR_CLKT = 0x02,Received Clock Stretch Timeout
  4. BCM2835_I2C_REASON_ERROR_DATA = 0x04, Not all data is sent / receive
  5. BCM2835_I2C_REASON_ERROR_TIMEOUT = 0x08 Time out occurred during sending
    Returns
    I2C error flag = 0x00 no error , > 0 bcm2835I2Creasoncode.

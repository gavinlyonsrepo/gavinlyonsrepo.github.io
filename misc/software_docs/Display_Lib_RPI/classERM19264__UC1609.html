begin Method initialise LCD Sets pinmodes and SPI setup

Parameters
AddressSetAC [2:0] registers for RAM addr ctrl. default=2 range 0-7
VbiasPOTcontrast default = 0x49 , range 0x00 to 0xFE
spi_dividerdefault = 64 ,see bcm2835SPIClockDivider enum , bcm2835
SPICE_Pindefault = 0 , which SPI_CE pin to use , 0 or 1

Start SPI operations(if HWSPI). Forces RPi SPI0 pins P1-19 (MOSI), P1-21 (MISO), P1-23 (CLK), P1-24 (CE0) or P1-26 (CE1) to alternate function ALT0, which enables those pins for SPI interface.

Returns
rpiDisplay_Return_Codes_e enum.
  1. Success rpiDisplay_Success
  2. Error 1 rpiDisplay_SPIbeginFail;

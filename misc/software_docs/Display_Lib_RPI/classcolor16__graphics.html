◆ spiWriteDataBuffer()

void color16_graphics::spiWriteDataBuffer ( uint8_t *  spidata,
int  len 
)
protected

Write a buffer to SPI, both Software and hardware SPI supported.

Parameters
spidatato send
lenlength of buffer
Note
The maximum size of an SPI transaction by default in lgpio library is 65536 or Display_SPI_BLK_SIZE. SO a buffer larger than this must be sent in blocks of 65536 bytes. 3 buffer blocks writes are supported So largest buffer that can be sent is Display_SPI_BLK_SIZE*3= 196608. The largest current expected size is for a full screen write to a 240x320 display currently which is 240x320x2 = 153600 bytes. spidev.bufsiz must also be set to 65536 or higher see Readme for display for more details.

◆ PixelFixMode_e

enum class GC9D01_TFT::PixelFixMode_e : uint8_t
strong

User-selectable workaround mode for narrow-window quirks on some GC9D01 displays i.e. missing pixels when drawing vertical lines or single pixel draws. behaviour may depend on display module variant and manufacturer.

Enumerator
Off 

Normal behavior – may show gaps, missing pixels

DoublePixel 

Apply double-pixel padding only in drawPixel

VFastOff 

Apply pixel by pixel only in drawFastVLine, no fast draw

Both 

Apply both fixes(default)

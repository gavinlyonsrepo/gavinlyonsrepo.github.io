Arduino based Micro-controller library list :

  1. 16-bit Graphics library
    • display16_LTSM: 16-bit Graphics Library
  2. 16-bit Displays
    • ST7735_LTSM: ST7735 TFT LCD
    • ST7789_LTSM: ST7789 TFT LCD
    • SSD1331_LTSM: SSD1331 OLED
    • ILI9341_LTSM: ILI9341 TFT LCD
    • GC9A01_LTSM: GC9A01 TFT LCD
  3. 1-bit Displays
    • ERM19264_UC1609: ERM19264 LCD UC1609C controller
    • ER_OLEDM1_CH1115 : ER-OLEDM1 Display CH1115 controller
    • NOKIA5110_TEXT: Nokia LCD 5110 PCD8544 display
  4. Seven Segment Drivers
    • TM1638plus: LED modules: TM163X & MAX7219
    • FourteenSegDisplay : LED segment displays
  5. Character LCD
    • HD44780_LCD: HD44780 LCD, I2C PCF8574 interface
  6. Sensors
    • MAX471: MAX471 current sensor module
    • BMP280_LTSM: BMP280 barometric pressure sensor
